MATLAB中<br>
* double()指将数据类型转换成浮点型<br>
* im2double()当输入数据为uint8时，函数输出为[0-1]的double型数据，相当于uint8数据除以255<br>
* mat2gray()也是输出数据为[0-1]的double型数据，只不过是根据输入矩阵的数据归一化<br>
